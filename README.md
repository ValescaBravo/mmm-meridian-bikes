# Marketing Mix Modelling (MMM) with Google Meridian — Bikes Retail Case Study

This repository contains a portfolio-ready Marketing Mix Modelling (MMM) case study built with **Google Meridian**.  
The goal is to estimate **incremental channel impact**, quantify **uncertainty**, and support **budget allocation** decisions using a Bayesian MMM workflow.

> Note: This is a demonstration project (dataset is not proprietary). Results are illustrative and are presented for methodology and decision-support storytelling.
